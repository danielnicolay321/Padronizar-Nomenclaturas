Padronizar-Nomenclaturas
============
Padronização de título de anúncios em modelos de item padrão.


Descripton
----------------
Ler o título de anúncio e escrever o modelo do item que está sendo vendido em uma nomenclatura padronizada baseado na marca/fabricante do item.


Required
-----------------
* [Microsoft.Office.Interop.Excel](https://www.nuget.org/packages/Microsoft.Office.Interop.Excel/)
